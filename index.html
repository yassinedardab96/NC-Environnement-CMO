🌿 Guide d'utilisation — Plateforme NC Environnement CMO

1. Qu'est-ce qu'une Non-Conformité (NC) ?

Toute situation observée sur le terrain qui ne respecte pas les exigences environnementales : déversement, déchets mal gérés, fuite, pollution, etc.

2. Comment déclarer une NC ?

  • Remplissez les champs dans l'ordre : Identification → Localisation → Classification → Constat → Plan d'action → Photos
  • Les champs avec propositions : cliquez pour voir les choix, vous pouvez en sélectionner plusieurs
  • Si votre choix n'est pas dans la liste : sélectionnez "Autre" et précisez
  • Cliquez Prévisualiser & enregistrer pour valider

3. Les photos

  • Ajoutez jusqu'à 2 photos Avant (situation constatée)
  • Ajoutez jusqu'à 2 photos Après (si la NC est corrigée)
  • Max 20MB par photo

4. Le registre

  • Visible en bas de page — toutes les NC soumises apparaissent en temps réel
  • Lecture seule — vous ne pouvez pas modifier les données

5. Fréquence de déclaration

  • Minimum 2 NC par mois par responsable
  • Toute NC observée doit être déclarée immédiatement, sans attendre

6. Niveaux de criticité

  • 🟢 Faible — impact limité, pas de risque immédiat
  • 🟡 Modérée — impact notable, nécessite une action corrective
  • 🔴 Critique — impact grave ou non-conformité réglementaire, action urgente requise

7. Le N° NC

Généré automatiquement — ne pas le modifier.

8. Conseils

  • Soyez précis dans la description
  • Ajoutez toujours une photo si possible
  • En cas de problème contactez : Yassine DARDAB — Responsable Environnement
